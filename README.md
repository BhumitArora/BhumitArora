### What I'm working on

Right now I'm building production data infrastructure at VortexifyAI — OAuth connectors, real-time streaming, event-driven microservices on Kubernetes. It's the kind of work where you learn fast because things break in interesting ways.

Before this I spent a summer at **Sprinklr** building a self-improving search agent using Knowledge Graphs, and did ML research at IIT Roorkee on stock forecasting with LSTMs and ensemble methods.

---

### Things I've built

**[RestroBot](https://github.com/BhumitArora/RestroBot)** — A RAG-powered chatbot with a two-stage retrieval pipeline (FAISS + BM25 fused via Reciprocal Rank Fusion), cross-encoder reranking, and LLM-as-judge evaluation built in. Uses Gemini 2.5 Flash for generation.

**[Finetuned Medibot](https://github.com/BhumitArora/Finetuned_Medibot)** — Fine-tuned DeepSeek-R1-Distill-Llama-8B on a medical QA dataset using LoRA adapters and custom chain-of-thought prompt templates.
